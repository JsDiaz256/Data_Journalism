Food Availability in NYC

Description of project:

This project will compare the amount of farmers markets, food carts, snack bars, and restaurants there are in different boroughs and zip codes in NYC. This project will show where the are the most businesses to grab a bite to eat in NYC.

Vocabulary:

  • Businesses or eateries - generic term to encompass all of the types of food-selling shops listed above

Prototypes:

Feedback:

  • "I think there may be too many graphs. The map does not feel very helpful on the first page, I would maybe use a dropdown instead. Or at least make the map show some data."
  • "Since the colors within the graph in the macro page are all red, I feel like different colors between the different factories would be visually better for the reader to see whats going on. I live the overall layout and how everything is very upfront and clear. However another piece of feedback would to be make the titles are little bigger. It would also be good if the grapes were organized clearer."

My prototypes were focused on finding different ways of comparing the boroughs and zip codes. Based on the feedback I got, I decided to reduce the amount of graphs, simplify them, and base them on easy-to-understand comparisons.

Data Sources:

  • NYC Farmers Markets
  • Directory of Eateries
  • Parks Zones

The Data Cleansing Pipeline:

  1. Download a few data sets from NYC Open Data
  2. Use Google Sheets to organize and remove unnecessary data
  3. Download the organized data as an .csv file
  4. Use a python program to read the .csv file and create a dictionary in a .json file
